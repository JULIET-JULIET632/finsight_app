# FinSight

Mobile-first Vite + React app to provide quick financial health insights for SMBs. Uses TailwindCSS.

Quick start:

1. cd into the project folder:

```powershell
cd Q:\Capstone\FinSight
```

2. Install dependencies:

```powershell
npm install
```

3. Run dev server:

```powershell
npm run dev
```

This project is scaffolded to live in `FinSight` under `Q:\Capstone` as requested.
